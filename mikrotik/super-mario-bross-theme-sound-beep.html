Super Mario Bross Theme Sound Beep - MikroTik Script RouterOS

This Super Mario Bross Theme Sound Beep script was made without calculating the actual notes (made off the top of my head without listening to the song recently), so it may sound a bit off on a few notes. The next version will have the fully calculated frequencies. I, Ian Harker, wrote this script May 13th and May 14th of 2008 during my spare time at the Chicago, IL MikroTik Essentials Training. The best way to add this script is to just copy/paste the code below into a new script (System -> Scripts -> +), give it a name, and then click "Apply":
